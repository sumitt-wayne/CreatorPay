# ⚡ CreatorPay

India's creator monetization platform. Accept UPI payments, automate Telegram access, and manage your entire subscriber base from one powerful dashboard.

---

## What is CreatorPay?

CreatorPay is built for Indian creators — educators, fitness coaches, stock analysts, anyone with a paid Telegram community.

**The problem:** Creators manually add subscribers after payment, manually remove them when plans expire. It wastes hours every week.

**The solution:** CreatorPay automates everything.

- Subscriber pays via UPI
- Bot instantly adds them to your Telegram channel
- Bot removes them when their plan expires
- You watch your dashboard and earn

---

## How it works

**For Creators:**
1. Sign up and create a subscription plan
2. Set your price, duration, and features
3. Connect your Telegram channel
4. Share your unique CreatorPay link

**For Subscribers:**
1. Visit the creator's page
2. Choose a plan and pay via UPI
3. Get instant Telegram access
4. Access expires automatically when plan ends

---

## Features

**Payments**
- UPI, cards, wallets, net banking via Razorpay
- Instant settlement to your bank account
- Automatic payment receipts

**Telegram Automation**
- Subscribers added instantly after payment
- Auto-removed when subscription expires
- One-time invite links for security

**Creator Dashboard**
- Revenue analytics with charts
- Subscriber management
- Active vs expired plan tracking
- Revenue by plan breakdown

**Subscription Plans**
- Create unlimited plans
- Monthly, quarterly, or yearly duration
- Custom features list per plan
- Enable or disable plans anytime

## Tech Stack

- **Frontend** — Next.js 15, Tailwind CSS, shadcn/ui
- **Database** — Supabase (PostgreSQL)
- **Auth** — Supabase Auth
- **Payments** — Razorpay
- **Automation** — Telegram Bot API
- **Animations** — Framer Motion
- **Hosting** — Vercel
